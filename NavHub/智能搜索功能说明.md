# NavHub 智能搜索功能说明

## ✨ 功能概述

已成功将 BinNav_Public 的智能搜索功能整合到 NavHub 项目中，提供更智能、更精准的搜索体验。

## 🎯 核心特性

### 1. **多关键词搜索**
- 支持空格分隔多个关键词
- 例如：`设计 工具` 会同时匹配包含"设计"和"工具"的内容

智能识别相关词汇，扩大搜索范围：

### 3. **相关性排序**
根据匹配程度自动排序搜索结果：

**权重规则**：
- 工具名称匹配：10 分
- 工具描述匹配：8 分
- URL 匹配：5 分
- 同义词关键词匹配（名称）：5 分
- 同义词关键词匹配（描述）：4 分
- 同义词列表匹配（名称）：4 分
- 同义词列表匹配（描述）：3 分
- 同义词列表匹配（URL）：2 分

得分越高，排名越靠前。

### 4. **智能提示**
搜索框提示文字：`🔍 智能搜索 - 试试输入'设计工具'或'前端开发'...`

## 🔧 技术实现

### 文件结构
```
NavHub/src/
├── utils/
│   └── smartSearch.ts          # 智能搜索核心逻辑
├── pages/
│   └── Home.tsx                # 集成智能搜索
└── app/components/
    └── header.tsx              # 更新搜索框提示
```

### 核心函数

#### `smartSearch(tools: Tool[], query: string): Tool[]`
主搜索函数，返回按相关性排序的工具列表。

#### `calculateRelevanceScore(tool: Tool, searchTerms: string[]): number`
计算工具的相关性得分。

#### `getSearchSuggestions(query: string): string[]`
获取搜索建议（可用于未来的自动补全功能）。
