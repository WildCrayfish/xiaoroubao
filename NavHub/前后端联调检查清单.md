## 🐛 常见问题排查

### 问题 1: 新增工具后刷新消失
**原因**: 
- Mock 数据模式未关闭
- 后端接口未正确保存数据
- 数据库未执行迁移脚本

**解决**:
1. 确认所有 API 文件的 `USE_MOCK = false`
2. 检查后端日志，确认数据已保存
3. 检查数据库，确认数据存在

### 问题 2: 跨域错误 (CORS)
**错误信息**: `Access-Control-Allow-Origin`

**解决**:
检查后端 `CorsConfig.java`，确保允许前端域名访问。

### 问题 3: 401 未授权错误
**原因**: Token 过期或无效

**解决**:
1. 重新登录获取新 token
2. 检查 `localStorage` 中的 token 是否存在
3. 检查后端 JWT 配置

### 问题 4: 500 服务器错误
**原因**: 
- 数据库连接失败
- SQL 语法错误
- 字段不存在（tags 字段未添加）

**解决**:
1. 检查后端日志
2. 确认数据库连接配置正确
3. 执行数据库迁移脚本
